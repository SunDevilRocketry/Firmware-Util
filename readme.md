## Firmware-Util

This project has been heavily vibe-coded using repl.it. Responsible AI usage is encouraged on this repository as a learning platform for new tools.


Unfamiliar with the firmware toolchain? Need a specific version of the firmware for testing or a launch? Use this utility to automatically install and flash the version you want to an SDR flight computer! Works on AMD64 windows computers. If you need support for another architecture, ping me on Slack.
